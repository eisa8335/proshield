ClickSend SMS Gateway

This application works with SMS Notification application.

Basic Features
  • Supporting application for "SMS Notification" module. Means this module is an add-on of Module "SMS Notification" . To use this module you must have installed first "SMS Notification"
  • This application provide required sms gateway facility to "SMS Notification" application for sending sms notification to the mobile number.
  • ClickSend Gateway module is able for sending text SMS directly to the customers,suppilers,buyers,any partners or any mobile number.
  • Send group sms or single sms using ClickSend sms gateway.
  • Track status of all sms send by ClickSend sms gateway.
