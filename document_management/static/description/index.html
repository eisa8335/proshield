Document Management System.


A document management system (DMS) is a system (based on computer programs in the case of the management of digital documents) used to track, manage and store documents and reduce paper.
In default odoo we dont have any DMS so we provide simple and very useful functionality to manage documents by this custom module.

New: Now we have added support to set department directly on directory, So you can add department on directory and users of that department will have directly access to related directory and documents.(In v10) So now you can share documents by Departments also very easily.

Directory Views

You can manage your documents by directory as normally all DMS do. You can share directory with the persons by adding them in users list. Users added to directory will have access to directory and realted documents of that directory.

Document Views

You can manage your documents with all necessary details as shown in form view. List view also gives required details. Access to any document can given to any user by adding perticular user as follower in relatedd document.

Tags Views

Document managers can manage tags from this view esily. You can give color on tags.

Search Views

In Document Management System when there are thousands of documents is is hard to find out realted document easily.
So to simplify this issue we have given proper search view on directory and documents views.
Search on tag and descripion is also possible.

Document User Management

Same as other main odoo applications this docuement management sstem also contains related users.
1. User: Can Add new documents and access directory and documents having access.
2. Manager: Can create, write and delete all documents, Directory and tags.
Note: Now we have added support to set department on directory, So you can add department on directory and users of that department will have directly access to related directory and documents(In v10).
